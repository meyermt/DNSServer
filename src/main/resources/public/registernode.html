Register Your Node

Instructions: PLEASE READ

Currently, you must follow these steps for registering a node:

1. Register superpeers first. Superpeers will register themselves first, and then other superpeers.

1a. A superpeer registering itself only needs to fill in node group (region), node name, and check the superpeer box.

1b. For each superpeer in the network, register their node group, name, ip, port, and check the superpeer box.

2. Register nodes in node group network. Visit this page from each node and register all information, but do not check superpeer.

3. Go back to the index, navigate to send a message to a peer in any connected network!
